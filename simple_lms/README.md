# Simple LMS System

A simple command-line Learning Management System (LMS) to manage student records.

## Features

- **Initial Data**: Starts with 5 pre-defined students.
- **Display Students**: View a list of all students currently in the system.
- **Add Student**: Add new students with Name, Email, Father's Name, and Semester.
- **Search Student**: Search for students by Name or Email.
- **Remove Student**: Delete a student record using their email address.

## How to Run

1.  **Navigate to the project directory**:
    ```bash
    cd simple_lms
    ```

2.  **Run the LMS**:
    ```bash
    python3 lms.py
    ```

## Usage

Upon running the script, follow the on-screen menu:

1. Display All Students
2. Add Student
3. Search Student
4. Remove Student
5. Exit
