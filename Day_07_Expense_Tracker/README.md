# Day 07 - Expense Tracker

## Project Description

The Expense Tracker is a Python command-line application designed to help users manage their daily expenses. It allows for adding expenses with specific categories and descriptions, viewing all recorded transactions, and providing a summary of total spending by category. Data is persisted in a `expenses.json` file.

## Features

- **Add Expense**: Record a new expense with amount, category, and optional description.
- **View Expenses**: List all recorded expenses with their details and timestamps.
- **Summarize Expenses**: View a total summary of spending grouped by category.
- **Persistence**: All expense data is saved to a JSON file, ensuring data is kept between sessions.

## How to Run

1.  **Navigate to the project directory**:
    ```bash
    cd Day_07_Expense_Tracker
    ```

2.  **Run the application**:
    ```bash
    python3 expense_tracker.py
    ```

## Usage

Follow the on-screen menu to interact with the application:
1. Add Expense
2. View Expenses
3. Summarize Expenses
4. Exit
